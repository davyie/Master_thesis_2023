## About The Project

This project is hosted by Svensk Kvalitetsindex and it explores the field of sentiment analysis, specifically clustering. This project will try to improve K-means algorithm by using domain knowledge and representation learning. 

The idea is that we have domain knowledge of how many clusters we want from the data and we know the labels for those clusters. We could then use representation learning of the labels to create the initial points and run K-means with those. This should improve the performance of the algorithm because we induce the algorithm to find clusters within those words. 

### Idea origin
This idea was raised because we know similar texts contains similar words or has similar semantic meaning. Humans can usually assign a single word to a set of sentences, e.g. grades, positive, very positive, negative etc. These words summerize the text into a single word and we could actually assign multiple different texts to these words.

## Dataset 
The project works with two datasets, one from Kvalitetsindex and one from ABSABank. Both of them are in swedish but the dataset from Kvalitetsindex has to be handled with care. 

### Svensk Kvalitetsindex 
This dataset contains data about the customers of Svensk Kvalitetsindex where they have information about product quality, service quality etc. This data cannot be publicly availabe and is therefore not imported in the repo. 

### ABSABank 
This dataset contains text from two news sites and one forum. This project will work on a subset of the texts which are opinions about immigration in Sweden. The data is labelled on different levels, paragraph, sentence and word. Their dataset is hosted on Språkbanken. The format of the annotations is WebAnno TSV3 and it originates from the tool WebAnno. 
[https://spraakbanken.gu.se/en/resources/swe-absa-bank]

## Getting Started

To start the project we have to enable the virtual environment for Python3 using venv. 
Go to the parent folder `./` where we should find a `.env` folder. This folder contains the virtual environment for Python3. We want to enable the enviroment by running the following command `source .env/bin/activate` and to deactivate the enviroment we run `deactivate` in the terminal.

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:davyie/Master_thesis_2023.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run main.py 
  ```sh
    python3 main.py
  ```
